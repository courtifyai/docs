Space collects the files you generated or worked with across your tasks, so you can find a result without reopening the task that produced it.

<Frame caption="Files collected in Space">
  <img src="/images/space-en.png" alt="Space showing generated files" />
</Frame>

## Find a file

- **Search** — search files by name.
- **Filter** — show **All** files or only your **Favorites**.
- **View** — switch between grid and list view.

Files a Worker generates during a task appear here automatically.

## Work with a file

- **Locate in task** — open the task that produced the file.
- **Download** — save the file to your device.
- **Share** — create a link to the file.
- **Copy path** — copy the file's path.
- **Favorite** — add a file to favorites so it is easy to find later.

<Tip>
  Mark deliverables you return to often as favorites, then use the Favorites filter to jump straight to them.
</Tip>

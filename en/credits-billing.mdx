CourtifyAI measures usage in credits. Your plan provides a credit balance, and Workers consume credits as they run.

## What uses credits

| Activity | How it consumes credits |
| --- | --- |
| AI activity | Conversations, reasoning, and task execution, based on the model and tokens used |
| Computer environment | Running a virtual computer environment, based on time used |
| File storage | Storing and accessing files, a small amount |
| Web search | Per query |
| People search | Per query |
| Third-party services | Based on the connected service |

Stronger models and longer, multi-step tasks use more credits than quick ones.

## Plans and trial

New accounts may include a trial period. From **Billing & Subscription** you can see your current plan, compare plans, and upgrade. A scheduled change takes effect on the date shown.

When you run out of credits, they refresh on the date shown on the billing screen, or you can upgrade to continue right away.

<Tip>
  To control spend, use the Adaptive run mode for lighter work and reserve Autonomous for deliverables that need deep, multi-step effort. See [Tasks](/en/tasks).
</Tip>

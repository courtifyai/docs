API 密钥让你从自己的脚本和应用中运行任务，而不必使用网页应用。创建密钥后，即可用它启动任务、查询状态、获取结果并发送追问。

## 创建密钥

<Steps>
  <Step title="打开 API 密钥">
    进入**设置**，打开 **API 密钥**。
  </Step>
  <Step title="创建并复制">
    创建新密钥并立即复制。请妥善保存，因为之后可能无法再次查看。
  </Step>
</Steps>

## 管理密钥

- **吊销** — 禁用密钥，使其无法再被使用。
- **删除** — 永久移除密钥。

密钥状态会显示为有效、**已吊销**或**已过期**。

## 用法示例

API 密钥界面提供常见操作的可直接复制的示例：

- 运行任务
- 查询任务状态
- 获取任务结果
- 发送追问

<Warning>
  请像对待密码一样对待 API 密钥。不要提交到代码仓库或分享给他人。一旦密钥可能泄露，请立即吊销。
</Warning>

任务是一次让 Worker 完成某件事的对话。每个任务都保留完整的历史记录、生成的文件和结果，你可以随时回到任务中继续。

## 开始一个任务

打开**新建任务**，在任务框中描述你想完成的工作。目标越清晰，结果越好，请说明输入材料、期望的产出以及返回格式。

上传文件或粘贴文本，给 Worker 提供可处理的材料，例如待审查的合同或待分析的数据。

## 选择模式

模式让 Worker 针对某类法律工作进行调优。选择与任务匹配的模式：

| 模式 | 适用场景 |
| --- | --- |
| 公司商事 | 公司与商事事务 |
| 知识产权 | 知识产权分析与侵权比对 |
| 不良资产 | 不良资产审查 |
| 合规审查 | 依据规则和内部规则进行检查 |
| 诉讼支持 | 案件准备与辅助分析 |
| 合同生成 | 生成与修改合同 |
| 法律演示 | 制作幻灯片和演示文稿 |
| 数据分析 | 分析结构化数据并生成图表 |

## 跟踪工作过程

任务运行时，你可以查看每个步骤、Worker 使用的工具（如网页搜索或虚拟电脑），并实时阅读结果。合同审查、侵权比对等子任务会在运行时显示出来。

## 完善与继续

在同一任务中回复，即可请求修改或追加工作。Worker 会保留之前的上下文，你无需重新开始即可迭代。Worker 生成的文件会收集在[文件空间](/zh/space)。

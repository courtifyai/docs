## 什么会消耗积分

| 活动 | 如何消耗积分 |
| --- | --- |
| AI 活动 | 对话、推理和任务执行，按模型和处理的 token 数计算 |
| 计算机环境 | 运行虚拟电脑环境，按使用时长计算 |
| 文件存储 | 存储和访问文件，少量消耗 |
| 网页搜索 | 按每次查询计算 |
| 人物检索 | 按每次查询计算 |
| 第三方服务 | 按所连接的服务计算 |

更强的模型以及更长、多步骤的任务，会比简单任务消耗更多积分。

## 套餐与试用

新账号可能包含试用期。在**计费与订阅**中，你可以查看当前套餐、比较套餐并升级。已排期的变更会在显示的日期生效。

积分用完后，会在计费界面显示的日期刷新，或者你可以升级以立即继续使用。

<Tip>
  为控制开支，较轻的工作使用「自适应」运行方式，把「自主」方式留给需要深度、多步骤投入的交付物。参见[任务](/zh/tasks)。
</Tip>
